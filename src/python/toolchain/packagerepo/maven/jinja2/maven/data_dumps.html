Data Dumps

Maven Artifacts

Download

A .csv file (with a header row). Each row consists of a pair id, coordinates, where id is a numeric id of some unversioned Maven artifact, and the coordinates are a colon-separated pair group_id:artifact_id.

For example:
1,org.apache.zookeeper:zookeeper
2,junit:junit
...

Maven Artifact Dependencies

Download

A .csv file (with a header row). Each row consists of a pair src_id, tgt_id, of Maven artifact ids, for which some version of the artifact src_id declares a dependency on some set of allowed versions of the artifact tgt_id. The dependency can be in any scope (compile, runtime, test etc.).

For example:
1,2
1,3
2,4
...
